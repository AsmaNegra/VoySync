# 🧳 VoySync

VoySync is a collaborative travel management platform designed to simplify trip planning and coordination.
The application enables users to organize journeys, manage itineraries, and collaborate efficiently within a group.

---

## 🚀 Key Features

* 🌍 Trip creation and management
* 👥 Collaborative planning with multiple users
* 📅 Itinerary organization and scheduling
* 🔎 Search and explore travel options
* 🔔 Notifications and updates for trip members
* 💾 Persistent data storage
